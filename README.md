# Messenger Tagger - NodeJs
### Compose a message through messenger using your credentials
### Customize mention tag

> We don't steal your credentials. Use it at your own risk!


## Install
> Requires nodejs@16^
```sh
npm install
```

## Run
```sh
npm start
```

## Compile
```sh
npm build
```

## Compatibility
```
Termux [nodejs-lts]
Windows
Linux
```
